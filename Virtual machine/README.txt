СИСНТАКСИС

// ввод
input IP1 - дает возможность ввести значение с консоли, которое записывается по указанному ip-адресу ячейки.

// вывод
cons IP1 - выводит значение, записанное по указанному ip-адресу ячейки.

// переменная
var IP ЗНЧ - записывает значение (2-й аргумент) в заданный ip-адрес (1-й аргумент).

// сложение
+ IP1 IP2 IP3 - значение, записанное в ячейке по 1-му адресу складывается со значением, записанным по 2-му адресу. Результат записывается в ячейку с 3-им адресом.

// вычитание
- IP1 IP2 IP3 - из значения, записанного в ячейке по 1-му адресу вычитается значение, записанное по 2-му адресу. Результат записывается в ячейку с 3-им адресом.

// умножение 
* IP1 IP2 IP3 - значение, записанное в ячейке по 1-му адресу умножается на значение, записанное по 2-му адресу. Результат записывается в ячейку с 3-им адресом.

// деление
/ IP1 IP2 IP3 - значение, записанное в ячейке по 1-му адресу делится на значение, записанное по 2-му адресу. Результат записывается в ячейку с 3-им адресом.

// деление с остатком
| IP1 IP2 IP3 - значение, записанное в ячейке по 1-му адрему делится на значение, записанное по 2-му адрему. Результат записывается в виде остатка деления в ячейку с 3-им адресом.

// прибавлени 1
++ IP - к значению, записанному в ячейке по указанному адресу, прибавляется 1

// вычитание 1
-- IP - из значения, записанного в ячейке по указанному адресу, вычитается 1

// присваивание 
= IP1 IP2 - значение, записанное в ячейке по 2-му адресу записывается в ячейку с 1-м адресом.

// сравнение 
== IP1 IP2 IP3 - значение, записанное в ячейке по 1-му адресу РАВНО значению, записанному в ячейке по 2-му адресу. Результат записывается в ячейку по 3-му адресу. Результатом может быть либо 0 - ложь, либо 1 - истина.
> IP1 IP2 IP 3- значение, записанное в ячейке по 1-му адресу БОЛЬШЕ значения, записанного в ячейке по 2-му адресу. Результат записывается в ячейку по 3-му адресу. Результатом может быть либо 0 - ложь, либо 1 - истина.
< IP1 IP2 IP3 - значение, записанное в ячейке по 1-му адресу МЕНЬШЕ значения, записанного в ячейке по 2-му адресу. Результат записывается в ячейку по 3-му адресу. Результатом может быть либо 0 - ложь, либо 1 - истина.
>= IP1 IP2 IP3 - значение, записанное в ячейке по 1-му адресу БОЛЬШЕ ИЛИ РАВНО значению, записанному в ячейке по 2-му адресу. Результат записывается в ячейку по 3-му адресу. Результатом может быть либо 0 - ложь, либо 1 - истина.
<= IP1 IP2 IP3 - значение, записанное в ячейке по 1-му адресу МЕНЬШЕ ИЛИ РАВНО значению, записанному в ячейке по 2-му адресу. Результат записывается в ячейку по 3-му адресу. Результатом может быть либо 0 - ложь, либо 1 - истина.
!= IP1 IP2 IP3 - значение, записанное в ячейке по 1-му НЕ РАВНО значению, записанному в ячейке по 2-му адресу. Результат записывается в ячейку по 3-му адресу. Результатом может быть либо 0 - ложь, либо 1 - истина.

// переход
go IP1 - переход к ячейке, адрес которой указан.

// условие
if IP1 IP2 - если значение, записанное в ячейке по 1-му адресу, истинно, то выполняется следующая строчка кода, иначе происходит переход к ячейке по 2-му адресу

// выход из программы
end - завершает выполнени программы.
